## Features
* Responsive layout
* High resolution
* Bootstrap CSS Framework
* Sass
* Gulp build
* AngularJS
* Jquery
* Charts
* Maps

## Build, Serve and Run

cd blur-admin
npm install

This will setup a working copy of BlurAdmin on your local machine

## Running local copy
gulp serve

This script should automatically open template in your default browser.

## Run local copy in production mode:
gulp serve:dist
